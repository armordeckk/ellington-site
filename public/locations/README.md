# Images des régions

Chaque ville de la page **Nos régions** (`/locations`) a son propre sous-dossier.

## Convention

- `<slug>/cover.jpg` — image principale (utilisée sur home + page Nos régions)
- `<slug>/01.jpg`, `02.jpg`… — galerie additionnelle (à activer plus tard)

## Sous-dossiers attendus

| Slug             | Ville                  |
| ---------------- | ---------------------- |
| `monaco`         | Monaco                 |
| `saint-tropez`   | Saint-Tropez           |
| `grimaud`        | Grimaud                |
| `cap-ferrat`     | Saint-Jean-Cap-Ferrat  |
| `cannes`         | Cannes                 |
| `sainte-maxime`  | Sainte-Maxime          |

## Format recommandé

- **Ratio** : 4:5 vertical
- **Dimensions** : ~1600 × 2000 px minimum
- **Format** : `.jpg` ou `.webp` (évite `.png` pour les photos)
